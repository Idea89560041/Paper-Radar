# Medical Imaging Deep Learning Paper Radar

This repository powers a daily GitHub Pages dashboard for recent deep learning and machine learning papers in medical imaging.

The radar focuses on neuroimaging, brain MRI/fMRI/PET, Alzheimer disease and dementia diagnosis, brain-gut axis and microbiome imaging, multi-organ diagnosis, medical image synthesis/enhancement, diffusion and generative models, foundation models, and vision-language or multimodal models for radiology.

The site is updated automatically every day at about 16:10 China/Hong Kong/Singapore time:

https://idea89560041.github.io/Paper-Radar/

It searches PubMed, arXiv, Semantic Scholar, and Crossref metadata from major journals and conferences, scores papers against the research profile in `config.yaml`, and publishes the highest-scoring matches as a static web page plus `papers.json`.

Traditional neuroscience papers without clear AI, machine learning, or deep learning method terms are filtered out.
